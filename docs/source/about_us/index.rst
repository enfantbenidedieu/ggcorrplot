========
About Us
========

ggcorrplot is an open source `Python <https://www.python.org/>`_ library dedicated to correlation matrix visualization using plotnine. It was created in :math:`2024`. ggcorrplot will always be :math:`100\%` open source software and free for all to use.
It is released under the liberal terms of the `MIT Licence <https://github.com/enfantbenidedieu/ggcorrplot/blob/main/LICENSE>`_. 

It is developed in the open on `GitHub <https://github.com/enfantbenidedieu/ggcorrplot>`_.
