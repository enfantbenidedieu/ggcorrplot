`ggcorrplot <https://github.com/enfantbenidedieu/ggcorrplot>`_ is an open source `Python <https://www.python.org/>`_ library distributed under 
the MIT Licence. It is used to visualize easily a correlation matrix using `plotnine <https://plotnine.org/>`_.
